### Run Locally

To run **BizTrack** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/richdede/BizTrack.git
```

Windows:

```bash
git clone https://github.com/richdede/BizTrack.git
```
